  1. How many countries does each continent have? List the continent name and the number of countries in a pie chart.

  2. For each continent, list how many countries it has in a pie chart.

VIS Query:

Visualize PIE SELECT T1.Continent , count(*) FROM CONTINENTS AS T1 JOIN COUNTRIES AS T2 ON T1.ContId = T2.Continent GROUP BY T1.ContId
