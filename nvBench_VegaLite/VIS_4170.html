NL Queries:

  1. Find the name of the project for which a scientist whose name contains ‘Smith’ is assigned to, and count them by a bar chart, and show by the how many name in descending please.

  2. Find the number of the name of the project for which a scientist whose name contains ‘Smith’ is assigned to, and show by the how many name in descending please.

  3. What is the name of the project that has a scientist assigned to it whose name contains 'Smith', and count them by a bar chart, and show by the how many name in descending please.

  4. A bar chart for finding the number of the name of the project for which a scientist whose name contains ‘Smith’ is assigned to, could you rank by the Y-axis from high to low?

VIS Query:

Visualize BAR SELECT T2.Name , COUNT(T2.Name) FROM assignedto AS T1 JOIN projects AS T2 ON T1.project = T2.code JOIN scientists AS T3 ON T1.scientist = T3.SSN WHERE T3.name LIKE '%Smith%' GROUP BY T2.Name ORDER BY COUNT(T2.Name) DESC
