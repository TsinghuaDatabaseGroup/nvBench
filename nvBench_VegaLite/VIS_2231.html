NL Queries:

  1. How many courses for each course description? Show me a stacked bar chart The x-axis is course description and group by office name, and order by the y-axis in descending.

  2. Show the number of courses for each course description in a stacked bar chart The x-axis is course description and group by office name, and rank from high to low by the y axis.

VIS Query:

Visualize BAR SELECT CRS_DESCRIPTION , COUNT(CRS_DESCRIPTION) FROM CLASS AS T1 JOIN employee AS T2 ON T1.prof_num = T2.emp_num JOIN course AS T3 ON T1.crs_code = T3.crs_code JOIN professor AS T4 ON T2.emp_num = T4.emp_num GROUP BY PROF_OFFICE , CRS_DESCRIPTION ORDER BY COUNT(CRS_DESCRIPTION) DESC
