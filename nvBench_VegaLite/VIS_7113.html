NL Queries:

  1. What is the number of each allergie that the student with first name Lisa has? Draw a bar chart, and could you list by the total number in desc?

  2. What is the number of each allergie the girl named Lisa has? Plot them as bar chart, and could you rank in desc by the y axis?

VIS Query:

Visualize BAR SELECT T1.Allergy , COUNT(T1.Allergy) FROM Allergy_type AS T1 JOIN Has_allergy AS T2 ON T1.Allergy = T2.Allergy JOIN Student AS T3 ON T3.StuID = T2.StuID WHERE T3.Fname = "Lisa" GROUP BY T1.Allergy ORDER BY T1.Allergy DESC
